## 怎么跑

前端是零依赖静态页，任意静态服务器即可：

```bash
cd week8-fullstack/src/frontend
python3 -m http.server 5173   # 然后浏览器打开 http://localhost:5173
```

后端另起（你 W2/W4 的服务）。接线时若遇跨域，在后端加 CORS 允许前端来源即可。
